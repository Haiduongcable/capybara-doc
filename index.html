Flexible Operation Modes

Choose the right mode for your workflow – from read-only planning to full autonomy.

Standard Mode

Balanced autonomy. Asks permission for sensitive actions.

capybara

Plan Mode

Read-only analysis and architectural planning.

capybara --mode plan

Safe Mode

Maximum control. Confirms every action.

capybara --mode safe

Auto Mode

Maximum autonomy for trusted workflows.

capybara --mode auto

capybara run "Create a python function that reverses a string"

Example: Adding a Custom Tool

Extend Capybara with your own async tools in just a few lines.

custom_tool.py 
from capybara.tools.registry import ToolRegistry

registry = ToolRegistry()

@registry.tool(
    name='sys_info',
    description='Get system information',
    parameters={'type': 'object', 'properties': {}}
)
async def sys_info():
    import platform
    return f'System: {platform.system()} {platform.release()}'
